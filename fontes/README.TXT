====================================
JOGO: FORCA MEMORIOSA
====================================

BEM-VINDO(A) AO JOGO!
Este jogo foi desenhado especialmente para exercitar a sua memória de forma fácil e divertida.

COMO JOGAR:
1. Toque em "START" para iniciar o jogo.
2. Uma dica ou charada aparecerá na tela.
3. Use o teclado virtual para adivinhar qual é a palavra escondida.
4. Se você errar muitas vezes, o bonequinho "cairá" e você perde a rodada.
5. Se acertar, você ganha pontos e vai para a próxima fase!

DICAS IMPORTANTES:
- Se achar as letras muito pequenas, clique em "CONFIGURAÇÕES" na tela inicial e arraste o controle para a direita para aumentar o botão e a tela inteira.
- Você pode desativar os sons nas configurações a qualquer momento.

Lembre-se de jogar com a tela do celular deitada (horizontal) para uma melhor visibilidade!

Boa sorte e divirta-se!
